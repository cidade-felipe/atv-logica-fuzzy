# Previsao de humor com logica fuzzy

Este projeto traz uma interface interativa para estimar o humor do dia usando logica fuzzy.

A versao atual usa Python para calcular a logica fuzzy e JavaScript apenas para manter a tela interativa. Os graficos sao desenhados localmente com Canvas, sem Plotly e sem depender de internet.

O resultado nao e diagnostico clinico. E uma leitura educativa baseada em regras fuzzy.

## Como usar

Rode o servidor local:

```powershell
python app.py
```

Abra no navegador:

```text
http://127.0.0.1:8000
```

Nao precisa instalar bibliotecas. O backend usa apenas a biblioteca padrao do Python.

## Como funciona

Fato: `app.py` serve os arquivos da interface e expõe a rota `POST /api/humor`.

Fato: o calculo fuzzy, a defuzzificacao, a classificacao e a geracao dos pontos do grafico ficam no Python.

Fato: o JavaScript envia os valores preenchidos para o Python, atualiza o resultado na tela e desenha o grafico local em Canvas.

Inferencia: manter a regra fuzzy no Python reduz duplicacao de logica e facilita testar o modelo sem depender do navegador.

Opiniao tecnica: usar um servidor Python minimo e melhor do que Flask neste caso, porque resolve o problema sem instalar dependencias e mantem o projeto leve.

## Limitacoes

- Nao substitui avaliacao psicologica ou medica.
- As regras foram definidas manualmente.
- A pontuacao e uma aproximacao didatica, nao uma previsao cientifica validada.
- Correlacao nao implica causalidade.
- Em uso real, seria necessario validar regras com dados reais, revisar vieses e tratar dados pessoais com cuidado.
